### 帳戶分工策略

- **TW_Brokerage** — 美股資本操作帳戶：短期交易、選擇權賣方策略、集中在高波動標的
- **Firstrade** — 美股配息帳戶：長期持有、收息、低周轉率
- **Cathay_US** — 定期定額 ETF（VOO, VTI, QQQ, BND）
- **IBKR** — 全球配置（ETF + 美股）
- **Yongfeng_A/B, Cathay_TW** — 台股長期持有 + 股息再投入

### 選擇權策略

- **主要策略**：Covered Call + Cash-Secured Put（賣方為主，收取權利金）
- **避險策略**：對核心持股買 Protective Put（尤其在高估值或總經風險升高時）
- **操作原則**：
  - DTE 30-45 天為主
  - 選擇 OTM 15-20% 的 strike
  - 當 profit > 50-75% 時考慮提前 close
  - 當 urgency = red 時必須立即處理（roll 或 close）
  - 單一標的曝險不超過組合的 15%

### 風險管理

- **單一個股最大虧損**：-20% 觸發減碼檢討，-30% 強制減半
- **整體組合最大回撤**：-15% 啟動避險（買 put、減少曝險）
- **避險工具**：SPY/QQQ put spread 作為組合保險
- **集中度警示**：單一標的超過 15% 組合比例時提醒分散

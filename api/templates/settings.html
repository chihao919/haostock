Google Sheets 連結
如何找到 Spreadsheet ID？ ▶
  1. 打開你的持倉 Google Sheet
  2. 網址格式為 https://docs.google.com/spreadsheets/d/1aBcDeFgHiJkLmNoPqRsTuVwXyZ/edit
  3. 粗體部分（/d/ 和 /edit 之間）就是 Spreadsheet ID
  4. 確保 Service Account 有此 Sheet 的檢視權限：
    • 在 Sheet 右上角「共用」→ 加入 Service Account email → 設為「檢視者」
LINE 通知設定
如何申請 LINE Bot？ ▶
  1. 前往 LINE Developers Console，用你的 LINE 帳號登入
  2. 建立一個 Provider（例如「投資助手」）
  3. 在 Provider 下建立 Messaging API Channel
    • Channel name: 任意（例如「投資 Agent」）
    • Channel description: 任意
  4. 進入 Channel 設定頁：
    • Messaging API 分頁 → 最下方 Channel access token → 點「Issue」產生 token → 複製貼到下方
    • Basic settings 分頁 → Your user ID（U 開頭）→ 複製貼到下方
  5. 用手機 LINE 掃描 Channel 的 QR Code 加 Bot 為好友（否則 Bot 無法推播給你）
Messaging API 分頁 → Channel access token（長字串）
Basic settings 分頁 → Your user ID（U 開頭，不是 Bot 的 ID）
Trello Focus Board
如何取得 Trello API Key 和 Token？ ▶
  1. 登入 Trello，前往 Power-Ups Admin
  2. 點「New」建立一個 Power-Up（名稱任意，例如「投資 Agent」）
  3. 建立後進入該 Power-Up → 左側選 API Key → 複製 API Key
  4. 在同一頁面，點擊 API Key 右側的 Token 連結 → 授權 → 複製產生的 Token
  5. 建立或選擇一個 Trello Board 作為 Focus Board
  6. Board ID 的取得方式：
    • 打開 Board → 網址格式為 https://trello.com/b/aBcDeFg/board-name
    • 粗體部分就是 Board ID

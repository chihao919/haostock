投資寶庫 Claude 設定指南

讓 Claude App（桌面版/手機版）和 Claude Code 直接查詢你的持股、分析策略、記錄交易

概覽

投資寶庫是一套基於 Google Sheets 的投資組合管理系統。你的持倉資料存在 Google Sheets，透過 API 或 MCP Server 讓 Claude 即時讀取。

前提：你需要一個 Google Sheets 存放持倉資料，以及本系統的 API Key。
如果你還沒有，請聯繫管理員取得。

方法一：Claude App（手機/桌面）推薦

透過 Remote MCP Server 連接，無需安裝任何東西。

1打開 Claude App 設定

桌面版：Settings → Integrations → Add Integration
手機版：左下角 Profile → Integrations → Add

2新增 MCP Integration

選擇 "Add custom MCP integration"，填入以下資訊：

Name:   投資寶庫
URL:    https://stock.cwithb.com/mcp

3授權連接

系統會自動導向 OAuth 授權頁面，點擊允許即可完成連接。

4開始使用

回到 Claude 對話，直接問：

"查詢我的美股持倉"
"NVDA 現在多少錢？"
"建議 Covered Call 策略"
"分析台積電基本面"

方法二：Claude Code（終端機）進階

適合開發者，透過本地 MCP Server 連接，功能更完整。

一鍵安裝

curl -sL https://stock.cwithb.com/setup.sh | bash

手動安裝

1下載 MCP Server

git clone https://github.com/chihao919/stock-api.git
cd stock-api/mcp-server
npm install

2設定 Google Service Account

到 Google Cloud Console 建立 Service Account，下載 JSON 金鑰，放到：

~/.config/claude-sheets/credentials.json

然後在 Google Sheets 中將 Service Account Email 加為編輯者。

3註冊 MCP Server

claude mcp add portfolio \
  -e PORTFOLIO_SHEET_ID="你的Spreadsheet_ID" \
  -e GOOGLE_SA_KEY="~/.config/claude-sheets/credentials.json" \
  -- node ~/stock-api/mcp-server/index.js

4啟動 Claude Code

claude
> 查詢我的美股持倉
> 幫我分析 NVDA 基本面

常見問題

Q: Claude App 連不上 MCP Server？
確認 URL 是 https://stock.cwithb.com/mcp（注意 https 和結尾的 /mcp）。如果之前連過，試著移除後重新加入。
Q: 手機版 Claude 也支援嗎？
支援！Claude iOS/Android App 都可以新增 MCP Integration，步驟跟桌面版一樣。
Q: 資料安全嗎？
所有資料存在你自己的 Google Sheets，API 透過 Service Account 讀取（唯讀或讀寫取決於權限設定）。API 端點需要 API Key 驗證。
Q: 可以多人使用嗎？
可以。每個用戶有自己的 Google Sheets，在 設定頁面 填入自己的 Spreadsheet ID 即可。
Q: 如何更新持倉資料？
直接修改 Google Sheets，或透過 Web 儀表板的 AI 對話功能上傳券商截圖自動解析。
